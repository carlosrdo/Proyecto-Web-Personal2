# Proyecto-Web-Personal2

En este ejercicio, solo se utilizarán HTML y CSS para crear una página web estática basada en la estructura descrita. El objetivo es construir el frontend de una web personal con un diseño sencillo que incluya una serie de características.

## Estructura del Proyecto

El sitio web personal contiene 4 páginas principales:

- **Inicio (index.html)** - Página de portada con bienvenida
- **Acerca de (acerca.html)** - Información personal y profesional
- **Portafolio (portafolio.html)** - Proyectos y tecnologías
- **Contacto (contacto.html)** - Formulario de contacto e información

## Características Implementadas

✅ **Menú de Navegación**: Enlaces a las cuatro páginas presentes en la parte superior
✅ **Cabecera**: Fondo con gradiente, título y subtítulo en cada página
✅ **Contenido específico**: Cada página tiene contenido único personalizado
✅ **Footer**: Pie de página común con enlaces a redes sociales y copyright
✅ **Diseño Responsive**: Adaptable a dispositivos móviles y tablets
✅ **Estilo Profesional**: Diseño limpio y moderno con CSS3

## Tecnologías Utilizadas

- HTML5
- CSS3
- Diseño Responsive (Mobile-first)
- Flexbox y CSS Grid
- Gradientes CSS
- Fuentes web

## Cómo Usar

1. Descarga o clona el repositorio
2. Abre `index.html` en tu navegador web
3. Navega entre las diferentes páginas usando el menú superior

## Vista Previa

El sitio web incluye:
- Navegación fija en la parte superior
- Headers con gradientes de colores
- Grid de elementos para portafolio y servicios
- Formulario de contacto funcional
- Footer con enlaces sociales
- Diseño completamente responsive

Desarrollado completamente con HTML y CSS, sin uso de JavaScript ni frameworks externos.
